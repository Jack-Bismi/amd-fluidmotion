# Contributing

Thanks for looking at this. I'm a solo dev working on this in my spare time, so any help is appreciated.

## What I need help with

- Testing on different AMD GPUs (I only have RX 7800 XT)
- ROCm compatibility fixes for RDNA2 cards
- Better optical flow models
- Documentation (I'm bad at writing docs)

## How to contribute

1. Fork it
2. Make your changes
3. Test on your AMD GPU (this is important — I can't test everything)
4. Open a PR with a clear description of what you changed and why

## Code style

No strict rules. Just make it readable. I use black for formatting but I'm not religious about it.

## Issues

If something doesn't work on your GPU, open an issue with:
- Your GPU model
- ROCm version
- Error output

I'll try to respond within a few days. No promises though.
